Про проєкт
Це платформа для цифрової ідентифікації Суверена на базі блокчейну Solana. Проєкт створений для реалізації концепції прямого народовладдя, де кожен громадянин може виступати законодавцем.
Основна мета
Цифрова ідентифікація: Використання блокчейн-технологій для захищеного підтвердження особи.
Громадяни як законодавці: Перехід від представницької демократії до прямого управління державою через цифрові інструменти.
Прозорість: Використання смарт-контрактів для гарантії незмінності волевиявлення.
Технологічний стек
Blockchain: Solana
Identity Provider: Reown (WalletConnect) AppKit
Frontend: HTML5, CSS3, JavaScript
